# Moghene Brand UI Mockups

This folder contains the approved visual moodboard and full-page UI mockups for the Moghene storefront and inventory admin.

## Visual direction

- Warm bone, near-black, indigo, burnt umber, oxblood, and muted brass
- Editorial display serif paired with restrained grotesk UI typography
- Nigerian textile craft shown through cinematic photography and tactile detail
- Gapless, asymmetrical layouts with clear commerce controls
- All existing catalog, filtering, sizing, cart, WhatsApp checkout, authentication, and inventory-management capabilities remain represented

## Files

- `00-moodboard.png` — approved brand and interface moodboard
- `01-storefront-home-full.png` — complete public homepage from hero through footer
- `02-storefront-shop-cart-full.png` — full catalog page with live inventory and open WhatsApp cart flow
- `03-storefront-lookbook-full.png` — complete immersive, shoppable lookbook
- `04-storefront-mobile-full.png` — complete responsive homepage and mobile cart state
- `05-admin-login.png` — inventory authentication
- `06-admin-inventory-dashboard-full.png` — complete product CRUD and storefront visibility workspace
